# Garmony (Songwriter)

Garmony is an end-to-end creative suite for songwriters and musicians. Understanding the vibe.

[![IMAGE ALT TEXT HERE](https://img.youtube.com/vi/G7SiExNki70/0.jpg)](https://www.youtube.com/watch?v=G7SiExNki70)

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
